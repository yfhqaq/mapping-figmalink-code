# Spotter Design System
欢迎使用 Spotter Design System，我们的目的是为更快、更好地交付高质量的 Spotter 品牌应用程序赋能。组件库包含以下内容

- [@spotter/ui](https://codeup.aliyun.com/spotter/spotter/ued/spotter-ui/spotter-desgin-system) - Spotter Design System 组件库
- [@spotter/desgin-token](https://codeup.aliyun.com/spotter/spotter/ued/atlas/tree/master/spotter-design-token) - Spotter Design Token, 主要包括颜色、字体、间距、阴影等设计变量

如果你有任何问题或疑虑，请随时联系我们 UX 团队。 我们的使命是为 Spotter 品牌应用程序提供一致的体验，使你的工作更加轻松、便捷。
